Modify the sketch with Claude Code

Project 1 • Light Signals
This is your first time using Claude Code as a pair programmer (this is called "Channel A Level 2"). You will describe a change you want to make to the sketch you uploaded in Milestone 2, and Claude Code will help you figure out what to change. Important: you describe the problem BEFORE you ask for a solution. That discipline is not optional — it is how Channel A Level 2 works in this program, and it is how you keep ownership of your own learning.

Step 1 — Fill in the (a)(b)(c) on this card before talking to Claude Code

Pick a change you want to make to your pattern. Easy changes to try: make it faster, make it slower, change which pin an LED is on, add a pause after every cycle.

(a) What I want to happen: _________________________________________________ _________________________________________________ (b) What is currently happening: _________________________________________________ _________________________________________________ (c) What I have tried (or looked at): _________________________________________________ _________________________________________________

Step 2 — Send the prompt to Claude Code

In the Claude Code window, paste this template (with your answers filled in), and also paste your current sketch code:

I am working on my Tier 2 Project 1 starter sketch. (a) What I want to happen: [your answer from Step 1] (b) What is currently happening: [your answer from Step 1] (c) What I have tried: [your answer from Step 1] Here is my current code: [paste the whole sketch file here] Can you help me figure out what to change?

Step 3 — Read Claude's answer, make the change, upload, test

  • Read Claude Code's response carefully. Do not just paste the code Claude gives you — understand which line is different from your old version.
  • Open your sketch in the Arduino IDE and make the change.
  • Click Upload.
  • Watch the LEDs. Did the behaviour change the way you wanted?
  • If not, go back to Step 1 with a new (a)(b)(c) and try again.

Comprehension check

Answer this in one sentence (out loud, or write it below):

After Claude Code helped me, the thing that changed in my sketch was: _________________________________________________ _________________________________________________
What you should see Your pattern is running differently from before — faster, slower, with different timing, or with different pin assignments. You can point at one line of code in your sketch and say "this is the line that changed." That is the Channel A Level 2 success state.
Done when
  • You filled in (a), (b), and (c) on this card before talking to Claude Code
  • Your pattern runs differently from the starter sketch
  • You can say in one sentence what Claude Code changed
Stuck? If Claude Code's answer did not work: don't panic. Go back to (a)(b)(c) and describe what happened MORE specifically. "It still blinks the same" is less useful than "the LEDs are now both on at the same time which is not what I wanted." See reference card R3 for more detail on how to prompt Claude Code well.
