💻הקוד שמעלים

03_blink_alternating.ino// Blinks two external LEDs in alternation.
// When one is on, the other is off.

const int LED_1 = 9;
const int LED_2 = 10;

void setup() {
  pinMode(LED_1, OUTPUT);
  pinMode(LED_2, OUTPUT);
}

void loop() {
  digitalWrite(LED_1, HIGH);
  digitalWrite(LED_2, LOW);
  delay(500);

  digitalWrite(LED_1, LOW);
  digitalWrite(LED_2, HIGH);
  delay(500);
}

שמים לב: בכל חצי של ה-loop יש שתי קריאות digitalWrite — אחת מאירה את לד 1 והשנייה מכבה את לד 2. אחר כך השהייה, ואחר כך ההפך. ככה "שני דברים בו־זמנית" עובדים בתוך loop אחד של ארדואינו.

👀מה אמורים לראות

שני הלדים מתחלפים לסירוגין — לד 1 דולק בזמן שלד 2 כבוי, אחר כך לד 2 דולק בזמן שלד 1 כבוי, וחוזר חלילה פעם בערך בשנייה.
