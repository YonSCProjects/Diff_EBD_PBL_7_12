Upload your first sketch

Project 1 • Light Signals
In this milestone you will upload new code to the Arduino that makes the "L" LED blink faster.

What to do

  • Open the file 01_blink_L_fast.ino in the Arduino IDE.The file is in your Project 1 folder under ino_files/01_blink_L_fast/.
  • Look at the Arduino IDE toolbar. Find the right-arrow button (→). This is the Upload button.
  • Click Upload. Watch the progress bar at the bottom of the IDE.
  • When upload finishes, you should see the green message "Done uploading" at the bottom.
  • Look at the "L" LED on the Arduino board. It should now be blinking FASTER than before — about three times a second.

The sketch you are uploading

This is the full code in 01_blink_L_fast.ino. You do not need to understand every line — just read it if you want to.

01_blink_L_fast.ino// Blinks the built-in "L" LED faster than the factory sketch.

void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);
  delay(150);
  digitalWrite(LED_BUILTIN, LOW);
  delay(150);
}
What you should see The green "L" LED on the Arduino board is now blinking about three times per second — noticeably faster than the factory blink from Milestone 1. The Arduino IDE shows "Done uploading" in green at the bottom. Your first Upload click worked.
Done when
  • The "L" LED is blinking faster than before
  • The Arduino IDE shows "Done uploading" in green
Stuck?
  • If Upload fails: check that Tools → Board shows "Arduino Uno" and Tools → Port shows a COM port.
  • If the IDE says "Arduino not found," unplug the USB cable, wait three seconds, plug it back in.
  • If still stuck after that, call the teacher.
