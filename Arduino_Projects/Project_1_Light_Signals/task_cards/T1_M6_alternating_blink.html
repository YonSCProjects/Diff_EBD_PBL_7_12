Upload the alternating-blink sketch

Project 1 • Light Signals
Now the Arduino controls two things at once. You upload a new sketch that blinks both LEDs — but never at the same time. When one is on, the other is off. This is your first time seeing the Arduino do more than one thing in the same sketch.

What to do

  • Open the file 03_blink_alternating.ino in the Arduino IDE. It is in your Project 1 folder under ino_files/03_blink_alternating/.
  • Click the Upload button.
  • Wait for "Done uploading" in green.
  • Watch your two LEDs. They should alternate — when LED 1 is on, LED 2 is off, then swap.

The sketch you are uploading

03_blink_alternating.ino// Blinks two external LEDs in alternation.
// When one is on, the other is off.

const int LED_1 = 9;
const int LED_2 = 10;

void setup() {
  pinMode(LED_1, OUTPUT);
  pinMode(LED_2, OUTPUT);
}

void loop() {
  digitalWrite(LED_1, HIGH);
  digitalWrite(LED_2, LOW);
  delay(500);

  digitalWrite(LED_1, LOW);
  digitalWrite(LED_2, HIGH);
  delay(500);
}

Notice something: there are two digitalWrite calls in each half of the loop. One turns LED 1 on while the other turns LED 2 off. Then a delay. Then the opposite. That is how "two things at once" works inside a single Arduino loop.

What you should see Your two LEDs alternate — LED 1 on while LED 2 is off, then LED 2 on while LED 1 is off, repeating about once per second. If both LEDs stay on all the time, or both stay off, or only one works, see the stuck section below.
Done when
  • Both LEDs are blinking in alternation
  • "Done uploading" showed in green in the IDE
Stuck? Both LEDs stay on all the time: one of them is probably wired to 5 V instead of to a digital pin. Check reference card R1 Wiring 2. Only one LED works: the other is wired backwards (flip it) or its resistor is not connected. Neither LED works: Upload may have failed — look for red error text in the IDE. Call the teacher if you cannot figure it out from R1.
