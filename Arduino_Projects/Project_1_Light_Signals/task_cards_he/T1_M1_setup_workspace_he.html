🏁פותחים את עמדת העבודה ומחברים את הארדואינו

פרויקט 1 • אותות אור

📋מה עושים

  • פותחים את סייר הקבצים של Windows. (Windows + E)
  • נכנסים לתיקייה: G:\My Drive\Arduino_Projects\
  • מוצאים את התיקייה שלכם, או יוצרים אותה יחד עם המורה.
  • בתוך התיקייה שלכם, יוצרים תיקייה חדשה בשם Project_1_Light_Signals.
  • קוראים למורה. המורה יפתח יחד אתכם את קלוד קוד ויכוון אותו לתיקיית Project_1_Light_Signals.
  • מחברים את הארדואינו Uno למחשב בכבל USB (הקצה המרובע לארדואינו, הקצה השטוח למחשב).
  • מוצאים על לוח הארדואינו את הלד הירוק הקטן עם האות "L" ליד רגל 13.
  • במחשב פותחים את Arduino IDE — הקיצור על שולחן העבודה.
  • בתפריט Tools בוחרים Board → Arduino Uno.
  • באותו תפריט Tools בודקים שב-Port מופיעה יציאת COM (למשל COM3 או COM4).
👀מה אמורים לראות

הלד הירוק הקטן עם האות "L" ליד רגל 13 מהבהב פעם בשנייה.
