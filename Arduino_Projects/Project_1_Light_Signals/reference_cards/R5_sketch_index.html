Which file to open at which milestone

Where your Project 1 files live

All your Project 1 sketches are in your Project 1 folder on Google Drive. The full path on the workshop PC is:

G:\My Drive\Arduino_Projects\<your_nickname>\Project_1_Light_Signals\ino_files\

Inside ino_files you will find seven sketch folders. Each folder contains one .ino file with the same name as the folder.

Open a sketch by double-clicking the .ino file, or by using File → Open in the Arduino IDE and pointing at the file.

Tier 3 — no starter sketch

At Tier 3 you design your own pattern from scratch with Claude Code Level 3. Your finished sketch lives in the same folder — pick any name you like (something like my_signature_pattern.ino).
