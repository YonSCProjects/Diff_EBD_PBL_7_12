Tier 2 • Milestone 1 of 5

Start-up — workspace, first LED, first upload

Project 1 • Light Signals
This milestone covers everything that Tier 1 did across Milestones 1 to 4, but moves faster because you are going to make design choices in the next milestones and the basics should not take a whole session. If you have never used the workshop before, this is still a together-milestone for the folder creation — the teacher will be with you for that part.

What to do

  • Workspace setup (together with the teacher if this is your first workshop session): open File Explorer → go to G:\My Drive\Arduino_Projects\ → create or find your folder → inside it create Project_1_Light_Signals → open Claude Code with that folder.
  • Plug in the Arduino Uno with the USB cable. Confirm the "L" LED on the board is blinking from the factory sketch.
  • Wire ONE LED on the breadboard: long leg → 220 Ω resistor → Arduino pin 9, short leg → GND. See reference card R1 Wiring 1.
  • Open ino_files/02_blink_external/02_blink_external.ino in the Arduino IDE. Click Upload.
  • Confirm the external LED is blinking once per second.
What you should see Your folder and Project 1 folder exist on Google Drive. Claude Code is open. The Arduino is plugged in. One LED is wired on pin 9 and blinking once per second. You are ready to make design choices in the next milestone.
Done when
  • Your folder and project folder exist
  • One LED wired on pin 9 and blinking
  • Claude Code is open and pointed at your Project 1 folder
