🏁מעלים את קוד ההבהוב לסירוגין

פרויקט 1 • אותות אור
עכשיו הארדואינו שולט בשני דברים בו־זמנית. מעלים קוד חדש שמהבהב את שני הלדים — אבל אף פעם לא יחד. כשאחד דולק, השני כבוי. זו הפעם הראשונה שרואים את הארדואינו עושה יותר מדבר אחד באותו קוד.

📋מה עושים

  • פותחים את הקובץ 03_blink_alternating.ino ב-Arduino IDE. הקובץ נמצא בתיקיית פרויקט 1 שלכם, בתוך ino_files/03_blink_alternating/.
  • לוחצים על כפתור ה-Upload.
  • מחכים להודעה "Done uploading" בירוק.
  • מסתכלים על שני הלדים. הם אמורים להתחלף לסירוגין — כשלד 1 דולק, לד 2 כבוי, ואז להפך.

💻הקוד שמעלים

03_blink_alternating.ino// Blinks two external LEDs in alternation.
// When one is on, the other is off.

const int LED_1 = 9;
const int LED_2 = 10;

void setup() {
  pinMode(LED_1, OUTPUT);
  pinMode(LED_2, OUTPUT);
}

void loop() {
  digitalWrite(LED_1, HIGH);
  digitalWrite(LED_2, LOW);
  delay(500);

  digitalWrite(LED_1, LOW);
  digitalWrite(LED_2, HIGH);
  delay(500);
}

שמים לב: בכל חצי של ה-loop יש שתי קריאות digitalWrite — אחת מדליקה את לד 1 והשנייה מכבה את לד 2. אחר כך השהייה, ואחר כך ההפך. ככה "שני דברים בו־זמנית" עובדים בתוך loop אחד של ארדואינו.

👀מה אמורים לראות שני הלדים מתחלפים לסירוגין — לד 1 דולק בזמן שלד 2 כבוי, אחר כך לד 2 דולק בזמן שלד 1 כבוי, וחוזר חלילה פעם בערך בשנייה. אם שני הלדים נשארים דולקים כל הזמן, או שניהם כבויים, או שרק אחד עובד, מסתכלים בחלק "תקועים" למטה.

🪄תקועים? שני הלדים נשארים דולקים כל הזמן: כנראה שאחד מהם מחובר ל-5 V במקום לרגל דיגיטלית. בודקים את כרטיסיית העזר R1 חיווט 2. רק לד אחד עובד: השני מחובר הפוך (הופכים אותו) או שהנגד שלו לא מחובר. אף אחד מהלדים לא עובד: ייתכן שה-Upload נכשל — מחפשים טקסט שגיאה אדום ב-IDE. אם לא מצליחים להבין מ-R1, קוראים למורה.
